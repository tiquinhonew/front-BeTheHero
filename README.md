# Be The Hero - Frontend

Projeto desenvolvido durante a Semana OmniStack 11.0 da Rocketseat. O objetivo é conectar ONGs que precisam de ajuda a pessoas que podem contribuir com tempo ou recursos.

## 🚀 Deployment

O projeto está disponível em: [https://front-be-the-hero.vercel.app](https://front-be-the-hero.vercel.app)

## 🛠️ Tecnologias

Este projeto foi atualizado para utilizar tecnologias modernas:

- **React 18**
- **Vite** (substituindo Create React App para melhor performance e segurança)
- **React Router Dom 5**
- **Axios**
- **React Icons**

## 💻 Como executar o projeto

### Pré-requisitos

- Node.js (versão >= 20.0.0 recomendada)
- npm ou yarn

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/tiquinhonew/front-BeTheHero.git
```

2. Instale as dependências:
```bash
npm install
```

3. Configure o arquivo `.env` com a URL da sua API:
```env
VITE_API_URL=https://sua-api.com
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

## 📦 Build

Para gerar a versão de produção:
```bash
npm run build
```
